# User App Portal

The primary user-facing application for end-consumers (e.g., dating/fitness users).

## Features
- **Profile**: Create and edit profile.
- **Discovery**: Swipe or search for other users.
- **Chat**: Real-time messaging.
- **Media**: Photo gallery uploads.

## Tech Stack
- **React 18**
- **Vite**
- **TypeScript**
- **Mobile-First Design**
